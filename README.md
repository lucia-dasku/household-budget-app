## Budget Planner Web App

A web application for managing personal or shared budgets with dynamic calculations and customizable plans.
The application allows users to create and manage budgets based on income, fixed expenses, and flexible spending categories. It supports different financial scenarios such as individual or couple budgeting.

### Development Note
This project is made in combination of my own knowledge supported by AI tools for learning, problem-solving, and improving code structure. 

### Features
- Add and edit income sources
- Manage fixed expenses and spending categories
- Automatic budget calculations
- Weekly and monthly breakdowns
- Save and load budget plans (local storage)
- Share budget via link (read-only mode)

### Tech Stack
- Next.js (React framework)
- TypeScript
- Tailwind CSS
- LocalStorage for persistence

### What I Learned
- Structuring a full-stack web application using Next.js
- Writing reusable logic with TypeScript
- Managing state and data flow in React
- Designing user-focused features for real-life financial use cases

### Getting Started

Clone the repository:
```
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

Install dependencies:
```npm install```

Run the development server:
```npm run dev```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
